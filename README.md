## 📌 Tentang Project

Personal **Web3 Developer Dashboard** milik **Imam Qolandani** (`dani12po`) — sebuah portfolio dashboard futuristik bertema cyberpunk yang dibangun dengan HTML murni dan di-deploy ke Vercel. Dashboard ini menampilkan profil, tech stack Web3, GitHub stats, repository terbaru, dan semua link media sosial dalam satu halaman yang interaktif.

---

## 🚀 Fitur Utama

- ⛓️ **Web3 Identity** — Branding sebagai Web3 Developer dengan tampilan cyberpunk
- 🎨 **Cyberpunk Dark Mode** — Tema gelap dengan aksen neon cyan & purple
- 🌐 **Particle Network Background** — Animasi jaringan node menggunakan Canvas API
- ✍️ **Typewriter Effect** — Cycling tagline: Web3 Developer, Open Source Builder, Decentralized Thinker
- 💎 **Glassmorphism Cards** — Card dengan efek blur dan neon glow
- 📊 **GitHub Stats** — Stats & contribution graph langsung dari GitHub
- 🗂️ **Dynamic Repos** — Fetch otomatis repository terbaru via GitHub API
- 🔗 **Social Links** — Semua platform media sosial dalam satu tempat
- 📱 **Fully Responsive** — Mobile, tablet, dan desktop friendly
- ⚡ **Zero Framework** — Pure HTML, CSS, dan Vanilla JS

## ⚙️ Cara Menjalankan Lokal

**Prerequisites:** Node.js

```bash
# 1. Clone repository ini
git clone https://github.com/dani12po/Dashboard.git

# 2. Masuk ke folder project
cd Dashboard

# 3. Install dependencies
npm install

# 4. Salin file env dan isi API key
cp .env.example .env.local
# Edit .env.local dan isi GEMINI_API_KEY dengan API key kamu

# 5. Jalankan development server
npm run dev
```

Buka browser dan akses `http://localhost:5173`

---

## 🌍 Deploy ke Vercel

```bash
# Install Vercel CLI (opsional)
npm i -g vercel

# Deploy
vercel
```

Atau cukup:
1. Push repo ini ke GitHub
2. Buka [vercel.com](https://vercel.com) → **New Project**
3. Import repo `dani12po/Dashboard`
4. Klik **Deploy** ✅
